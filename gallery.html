Enhancement One: This was my work completed initially for CS330 in early 2025. We were tasked with creating a basic scene using open GL and basic geometry. In my initial assignment for this course I had nothing more than a few balls and boxes sitting on a single plane. When diving back in for my initial enhancement, I sought to further expand this scene. I did so mby rendering a full desk for my PC to sit on. I added walls and floors, all of which were textured properly, and even constructed a microphone for my desk to more accurately represent the setup in which I was on at the time of creation.

Enhancement Two: This enhancement was completed for my work done back in my first term at SNHU in 2023 for CS210. This was a simple grocer app meant to read from a text file to display data in a compact format. With my enhancement I sought to add additional deatures to this grocer app to act and serve as both an inventory app and a real-time tracker. With this enhancement I added the ability to add items to inventory in bulk when separated by commas, and remove items from the inventory on a one-by one basis, as well as a real-time tracker to show how many of each item was in stock at any given time. I also added a search feature to make it easier to find specific items in the inventory.

Enhancement Three: This enhancement was completed for my final project done in my CS340. The initial project was a database for an animal shelter meant to search for disaster rescue dogs. In my enhancement, I went in and added security measures for users logging in, setting permissions for what unregistered users were allowed to edit or see. A register option was also added to allow for new users to be added to the system. With this, I also added password hashing, allowing a users password to be hashed before being stored in the database for added security. This was the focus of the enhancement, as the database already functioned well for the search aspects.
